## Development

I'd recommend you use vue-cli tools for better experience. More info [here](https://cli.vuejs.org/guide/#cli).

```
npm i -g @vue/cli
vue ui
```

#### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Runs tests
```
npm run test:unit

npm run test:e2e
```
